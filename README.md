Welcome to my personal Machine Learning and Data Science portfolio! This repository serves as a centralized hub for my independent projects, showcasing my ability to solve diverse, real-world problems using artificial intelligence. 

## 🚀 What's Inside?

This repository is designed to grow. Rather than focusing on a single niche, it spans multiple domains of Machine Learning—from structured business data to unstructured image processing. 

Each major project lives in its own dedicated folder, complete with a specific `README.md` detailing the methodologies, code, and business insights.

### 📊 Predictive Analytics & Tabular Data
Projects focused on applying statistical rigor and advanced machine learning to structured data to drive business decisions.
* **Featured Project:** *Supply Chain Delay Prediction*
* **Core Skills:** End-to-end ML pipelines, strict prevention of data leakage, feature engineering, and optimizing models (Random Forest, XGBoost) for specific business metrics like Precision vs. Recall.

### 👁️ Computer Vision & Deep Learning
Projects focused on teaching machines to interpret, process, and classify unstructured visual data.
* **Featured Project:** *Handwriting Recognition Pipeline*
* **Core Skills:** Image preprocessing (grayscale, binarization), pixel normalization, and building Neural Networks (CNNs) to classify complex spatial patterns.

## 🛠 Core Tech Stack
Across the various projects in this repository, you will find implementations utilizing:

* **Languages:** Python
* **Data Manipulation & Analysis:** Pandas, NumPy, SciPy
* **Machine Learning:** Scikit-Learn, XGBoost, Support Vector Machines (SVM)
* **Deep Learning:** TensorFlow / PyTorch
* **Data Visualization:** Matplotlib, Seaborn

## 📂 How to Navigate

Feel free to explore! To view a project:
1. Click into any of the specific project directories above.
2. Read the project-specific `README.md` for a high-level overview of the results.
3. Open the Jupyter Notebooks (`.ipynb`) to view the step-by-step code, exploratory data analysis, and model training processes.
