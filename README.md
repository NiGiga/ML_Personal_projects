## 🚀 Highlight Project: Supply Chain Delay Prediction

The main project currently in this repo focuses on predicting **Late Delivery Risks** in supply chain operations.

### 📝 Project Overview
The goal is to build a robust model that can identify orders at risk of delay before they are shipped. This allows for proactive logistics management and improved customer satisfaction.

### 🧪 Key Methodologies Applied:
* **Data Cleaning & EDA:** Comprehensive analysis of distributions, skewness, and outliers.
* **Statistical Outlier Handling:** Used Z-Score for normal distributions and IQR for skewed features (e.g., Product Price, Sales).
* **Feature Engineering:** Extracted temporal patterns (Order Month, Day of Week) and handled categorical variables via One-Hot and Ordinal Encoding.
* **Advanced Pipelines:** Implemented Scikit-Learn `ColumnTransformer` and `Pipeline` to prevent Data Leakage.
* **Model Benchmarking:** * **Random Forest:** Achieved ~75% accuracy with strong precision for late deliveries.
    * **XGBoost:** Explored gradient boosting for complex pattern recognition (82% precision on delays).
    * **SVM:** (Ongoing) Testing non-linear boundaries for optimal separation.
