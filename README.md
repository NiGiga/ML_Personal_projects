## 📁 Projects Included

### ✍️ 01. Handwriting Recognition (MLP)
**Directory:** `01_handwriting_recognition/`
**Main File:** `handwriting_recognition_model.ipynb`

In this project, I developed a neural network model to recognize handwritten digits using standard data science libraries.

* **Dataset:** Built using the Scikit-Learn `digits` dataset (8x8 pixel images).
* **Architecture:** Implementation of a **Multi-Layer Perceptron (MLP)** with a Stochastic Gradient Descent (SGD) optimizer and logistic activation function.
* **Performance:** The model achieves approximately **94% accuracy** on the test set.
* **Validation:** Robustness was verified using **K-Fold Cross Validation** (k=5).
